# Ejecutar en desarrollo

1. Clone respository
2. Ejecutar
```
npm install
```
3. Have nest/cli installed
```
npm i -g @nest/cli
```
4. Start database
```
docker-compose up -d
```


## Stack usado
* MongoDB
* Nestjs
* Html 5
